# ASP.NET CRUD Product Application

This is a **CRUD (Create, Read, Update, Delete) Product Management** application built with **ASP.NET Core 8.0** and **Entity Framework Core**.  
It provides a RESTful API to manage products and can be used with any frontend, including Flutter, React, or Angular.

## Features

- List all products  
- Get product details by ID  
- Create a new product  
- Update an existing product  
- Delete a product  
- Works with both HTTP and HTTPS (for emulator and mobile apps)  
- Supports sorting, e.g., latest products on top  

---

## Technologies

- ASP.NET Core 8.0  
- Entity Framework Core  
- SQL Server / LocalDB / MariaDB (any supported DB)  
- RESTful API  
- JSON for data exchange
